How to use:

  1. Copy the path of the folder that contains the files that you want to rename.
    You can do that with right click on the Explorer path.

  2. Select which symbol or character do you want to remove from your files names.

  3. Select what do you want instead of previous symbol.
    This Character will replace all characters that you chose in step 2.

  4. Pres the button and watch the magic happen!

That's all folks, happy renaming!
